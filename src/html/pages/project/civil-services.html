Civil Services

An Open Source Project

Civil Service USA Corp is a nonpartisan, independent and Non-Profit Organization with the goal to help United States Citizens to be a part of what is happening in their Local, State & Federal Governments.

While working on several custom projects for StayWoke.org and CampaignZero.org, we discovered a need for a more robust API to help United States Citizens be a part of what is happening in their Local, State & Federal Governments. We ended up creating a massive API and a full-featured website that acts as a central location for accessing information about elected officials.
